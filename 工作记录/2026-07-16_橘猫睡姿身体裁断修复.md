# 2026-07-16 橘猫睡姿身体裁断修复

## 现象

- 暖橘皮肤处于日常睡眠状态时，左侧后半身出现垂直缺口，看起来像图片少了一块。

## 根因

- 暖橘 4×2 原始图集中，右下角睡姿的后半身向左跨出了自己的网格单元。
- 拆图脚本严格按 384px 单元格裁剪，把越界的身体切掉，形成长直线断面。
- 旧测试只检查图片尺寸、透明通道、覆盖率与连通性；被裁断的图片仍是单一连通体，因此没有被发现。

## 修复

- `工具/prepare_cat_sprites.py` 为睡姿单元格增加 48px 左侧安全重叠区。
- 从 `assets/skins/orange/source.png` 重新生成暖橘 `sleep.png`。
- 增加 alpha 左边缘长直线检查，防止主体再次被网格边界裁断。
- 增加无摄像头橘猫睡姿启动预览。

## 验收

- 旧素材左边缘不透明占比为 45.28%，新回归测试可稳定捕获。
- 新素材针对性测试 2 / 2 通过。
- Python 语法检查通过。
- 全量自动化测试 119 / 119 通过。
