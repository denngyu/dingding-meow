# 2026-07-15 · 图标统一：托盘/exe/自启用同一只健康报告猫头

## 需求

用户截图指出：健康报告浏览器标签页的猫头（`report_icon.py` 里的 favicon）好看，希望**所有的图标**都用它。

## 现状盘点

- **健康报告 favicon**：`report_icon.py` 从 `assets/cat_sprites/idle.png` 裁上 66% 得到猫头，缩到 58×58 居中放进 64×64 透明画布，Base64 嵌入 HTML `<link rel="icon">`。视觉一致的**基准**。
- **托盘图标**：`pet.py::make_tray_icon` 手绘 PIL 三角耳 + 椭圆头 + 两颗黑眼，跟报告 favicon 是两只不同的猫。
- **exe 图标**：v1.0 打包没传 `--icon`，Windows 显示 PyInstaller 默认蓝盒子。
- **开机自启 .lnk**：`set_autostart` 只设 TargetPath 到 .vbs，IconLocation 走 Windows 默认（.vbs 关联的 wscript 图标）。

## 验收标准

- **AC1** 托盘弹出栏里的小猫图标肉眼就是报告 favicon 那只
- **AC2** File Explorer 里 `盯盯喵.exe` 显示同一只猫头（不再是 PyInstaller 默认蓝盒子）
- **AC3** 勾了「开机自启」后，`%APPDATA%\...\Startup\盯盯喵.lnk` 显示同一只猫头
- **AC4** `report_icon.py` 只保留一份裁剪逻辑，不同尺寸/用途共用；favicon 与 tray 像素完全一致
- **AC5** `assets/dingdingmeow.ico` 是多尺寸 (16/24/32/48/64/128/256) 的 Windows ico，随源码提交、随便携包发布
- **AC6** 从源码跑（无 exe）也能用：`set_autostart` 找 `assets/dingdingmeow.ico` 设 IconLocation

## 实现

### `report_icon.py` 重构
- 抽出 `build_cat_head_image(size, source_path)` 作为通用函数，支持任意 size
- `build_report_favicon_image(source_path)` 退化成 `build_cat_head_image(64, ...)` 的薄壳
- 新增 `save_ico(path, source_path, sizes=ICO_SIZES)` 一步生成 Windows 多尺寸 .ico
- Fallback 手绘头像也做成按 size 缩放（`_fallback_cat_icon(size)`），保持一致体验

### `pet.py`
- `make_tray_icon()` → `return report_icon.build_cat_head_image(64)`（3 行，从 10 行手绘缩到 1 行调用）
- `set_autostart(True)` → 若 `assets/dingdingmeow.ico` 存在，往 powershell 脚本里追加 `$l.IconLocation = ...`；找不到 ico 就走默认（不崩）
- `import report_icon` 顶部加进去

### 图标文件
- 一次性 `python -c "import report_icon; report_icon.save_ico('assets/dingdingmeow.ico')"` 生成 104KB 多尺寸 ico，进 git

### 测试
- `tests/test_report_icon.py`：新增 `build_cat_head_image` 多尺寸 / 居中带边距 / favicon 与 head 像素一致 / `save_ico` 生成合法 ICO 四个测试
- 现有 favicon 两个测试保持不变
- `python -m unittest discover tests` 全过 64/64

### 打包
- `pyinstaller ... --icon assets\dingdingmeow.ico ...` 把 ico 嵌进 exe
- 便携 ZIP 需要把 `assets/dingdingmeow.ico` 复制到 exe 旁边，`set_autostart` 才能引用到

## 不做

- 不改新手教程窗口图标（`onboarding.py` 是 Toplevel + overrideredirect，无系统级图标位）
- 不改水弹窗的 💧 emoji 头图（那是内容装饰不是应用图标）
- 不清 `dist_v14/`、`dist_v1old/` 归档，只重建 `dist/`
