# v1.4 稳定版发布

## 发布范围

- GitHub Pages 宣传页：`promo.html` 与 `index.html`
- 单页说明：`onepage.html`
- 首次启动教程：`onboarding.py`，教程版本升级到 2
- GitHub 项目说明：`README.md`
- Windows 便携版：`DingDingMeow-windows.zip`
- GitHub 公开源码与 Release：`v1.4.0`

## 发布功能

- 约 81px 独立灰白毛球沿 X 轴缓慢滚到中央，再滚回原位
- 中央叩屏三次，删除玻璃凹陷与波纹
- 三瓣蒸汽缓慢淡入淡出并轻微上飘
- 锁屏立即离开，摄像头无帧继续推进离开计时
- 短暂丢脸 30 秒内保留久坐会话
- 双正脸分类器交叉确认与侧脸对称假目标过滤
- 饮水误判支持 Esc 和右上角关闭
- 新手教程补充中央提醒、锁屏和 30 秒续接说明

## 发布边界

- 不发布 `walk`、`wake` 与日常逐帧呼吸实验
- 不提交 `logs/`、`settings.json`、历史 `dist/` 和本机构建缓存
- 发布包仅包含 EXE、运行素材、YOLO 模型、启动器和使用说明

## 验证

- 稳定版自动化测试：97 项全部通过
- Python 语法检查通过
- `index.html` 与 `promo.html` 字节一致
- EXE 构建、便携目录启动和 ZIP 内容检查在发布阶段记录
