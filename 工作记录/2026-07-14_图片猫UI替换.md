# 图片猫 UI 替换

日期：2026-07-14

## 目标

把用户选定的上一版灰白猫图集接入盯盯喵源码 UI，并让宣传页使用同一角色形象。保留现有窗口尺寸、气泡、摄像头面板、业务状态和提醒节奏。

## 素材处理

- 原始素材保存为`assets/cat_sprite_source.png`
- 使用`工具/prepare_cat_sprites.py`把4×2图集拆成8张图片
- 对每格执行边缘连通背景去除、主体裁切、等比缩放和透明画布居中
- 运行素材统一为384×384 RGBA PNG，放在`assets/cat_sprites/`
- 8个状态：`idle`、`watch`、`tail`、`over`、`hold_cup`、`drink`、`away`、`sleep`

## UI 接入

- 新增`cat_sprites.py`，集中管理资源路径、状态到图片的映射和眼睛点击热区
- `pet.py`启动时一次加载8张图片并缓存`ImageTk.PhotoImage`
- 图片猫显示尺寸为144×144，维持现有236×250闭合窗口构图
- 久坐继续使用低幅局部位移；喝水仍在2.4秒动作完成后弹记录框
- `promo.html`首屏改用`assets/cat_sprites/watch.png`，增加2px低幅呼吸动画并支持`prefers-reduced-motion`
- 旧`cat_visual.py`保留，用于连续姿态采样和历史测试；主UI不再调用其矢量猫绘制

## 验证

- `python -m unittest discover -s tests -v`：12项通过
- `python -m py_compile pet.py cat_sprites.py cat_visual.py 工具/prepare_cat_sprites.py`：通过
- Tk隐藏窗口冒烟：8个PIL资源、8个`PhotoImage`全部加载成功
- 主模块导入：`IMPORT_OK 144`
- 视觉预览检查：静坐、观察、久坐、喝水四种状态在实际窗口比例下无裁切，气泡与猫的视觉比例正常

## 发布说明

- 本次只修改源码和宣传页，没有改`dist/`、`dist_v1old/`或`版本存档/`
- 当前`dist/`仍是v1.3旧UI
- 下次重打exe时必须把`assets/cat_sprites/`复制到exe同级目录；原始图集和拆分脚本不用随包发布
