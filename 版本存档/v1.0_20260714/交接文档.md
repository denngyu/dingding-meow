## 一句话
桌面小猫浮窗,用电脑摄像头管你「久坐 / 离座 / 喝水」,纯本地、免费。技术栈 Python + OpenCV + YOLOv4-tiny + tkinter。

## 项目位置
`D:\AI\claude code\本地之眼\` (产品名已改叫**盯盯喵**,文件夹名暂未改)

## 文件结构
- `pet.py` — 主程序,**全部功能都在这一个文件**(检测循环 + tkinter浮窗 + 喝水弹窗)
- `宠物浮窗.vbs` — 无黑窗启动器(日常双击这个)
- `宠物浮窗.bat` — 备用启动(pythonw)
- `models\` — YOLO模型(yolov4-tiny.cfg/weights),喝水的杯子检测用
- `dist\本地之眼.exe` + `dist\models\` — 打包好的exe版(双击即用,不装python)
- `logs\` — 每日日志(sit_日期.csv久坐/离座, water_日期.csv喝水)
- `promo.html` — 宣传页 / `项目复盘.md` — 复盘 / `版本存档\` — 历史版本

## 怎么跑
- 源码: 双击 `宠物浮窗.vbs`
- exe: 双击 `dist\本地之眼.exe`(exe要和旁边models文件夹在一起)
- 依赖: `pip install opencv-python==4.10.0.84 pillow numpy`

## 五个必须知道的坑(血泪)
1. **opencv 必须 4.10**,5.0删了 `cv2.CascadeClassifier`(人脸检测)。别升级。
2. **opencv 不认中文路径**:加载YOLO模型要先 `shutil.copy` 到系统temp英文路径;若要读写图片用 `imencode+tofile`/`imdecode+fromfile`。
3. **打包**: `pyinstaller --onefile --windowed --collect-all cv2 --collect-all PIL`,models不打进exe、放exe旁边(pet.py用exe目录找models)。
4. **无黑窗**:靠 .vbs `WScript.Shell.Run "...pythonw... pet.py",0` 启动;双击.bat会闪cmd。
5. **验证脚本输出**加 `python -u`,否则强杀进程时print缓存丢失、误判没跑。

## 检测逻辑要点(在 pet.py 顶部可调参数)
- 判在座: 人脸检测(default+alt, minNeighbors=5, minSize=80, 收紧防背景误检) + **宽限期**GRACE_SEC(防托腮偶尔漏检); **over状态用OVER_GRACE=5秒**快速认离开
- 遮挡: 画面平均亮度 < BLOCK_LEVEL 判被捂
- 喝水: YOLO检测 cup/bottle/wineglass(class 39/40/41, CONF=0.25) 的框与脸框重叠(near函数, 脸框放宽) → 弹窗记ml
- 摄像头**常开**(cap不每轮开关)提速;GUI用tkinter+PIL显示帧
